My Story

2019: The Start

I first knew about coding when I watched a youtube video on game design.The youtuber talked about how he created a romhack (modifying the code of a game to create a different game experience) when he was younger. This inspired me to want to learn coding. To be able to create a game.

2020-2021: Javascript

I started with javascript using Grasshopper as I read online that javascript was curcial to creating an interactive game. I completed the fundamentals

2021: Java crash course

I attended a one week online java crash course. It was the first time I learnt java and it was extremly helpful. I learnt both the basics and advanced Java using Oracle and NetBeans.

2021: Java

The first java programm I learnt how to make was a calculator. It was also the first time I did intense coding for so long. It showed me the dedication and perserverace needed in order to suceed in coding.

2021-present

After that I continued doing personal coding projects. I learnt both HTML and CSS by following youtube videos and coding websites such as the one you are seeing right now.

2022: HTML, CSS and website creation

Learning HTML and CSS was surprisingly easy and I really enjoyed seeing my website come to life. Going through the process of coding the website gave me a sense of accomplishment and I learnt to trust the process along the way.

2022: YCEP

In 2022, I took part in the Youth Cyber Exploration Programme (YCEP) hosted by NYP. It opened my eyes to the world of hacking and the vulnarabilities of software and the need to protect them.

2022: What I learnt in YCEP

YCEP taught me many things such as basic python, Linux and cryptography. I took part in my first Capture the Flag(CFT) competition and enjoyed myself immensly.

2022: My first CFT

I really enjoyed the CFT as it made me think outside the box and try out solutions I would have never thought of before. It allowed me to look at the things in a new perspective and was very satisfying when I finally solved the puzzles.

Present: Thoughts, Regrets and Aspirations

One thing that bothered me while doing my EAE is the lack of certificates. I only attended 2 course and attained 2 certificates over the 3 years. However, I am sort of grateful that I did not join more courses as it truly allowed me to grow and learn at my own pace. The more I code things by trial and error and searching the internet for answers the more my passion for coding grew. In the future I want to have a greater understanding in the wide and complex world of technology and to work and excel in this field. Thank you for reading.
